# Purpose

- Can be use out of the box in X86_64 linux, Just throw into one directory, and run it.

# Usage

1. Download package.

```sh
$ git clone https://github.com/zw963/linux_key_rebinding
```

2. Edit your's config in $HOME/.rbindkeys.rb

3. Start as daemon.
